# Contributing to Awesome AI-driven Plant Biology

Thank you for your interest in contributing to **Awesome AI-driven Plant Biology**!

This repository aims to maintain a curated and structured collection of research papers on **AI-driven methods in plant science**.  
We welcome contributions from the community to help keep the resource **up-to-date, comprehensive, and well-organized**.

---

## How You Can Contribute

You can contribute in several ways:

- Suggesting **new research papers**
- Recommending **review articles**
- Improving the **taxonomy or category structure**
- Fixing formatting or metadata issues
- Suggesting **important AI methods applicable to plant biology**

---

## Submitting a Paper

If you would like to suggest a new paper, please include:

- **Paper title**
- **Authors**
- **Year**
- **Conference or journal**
- **Link to paper** (preferably DOI, arXiv, or publisher page)
- **Short description** (1–2 sentences)
- Suggested **category**

## How to Contribute

We welcome contributions from the community. You can add new papers, improve the organization, or fix formatting issues.

The standard contribution workflow is shown below.

1. Fork this repo on your personal GitHub.

    Click the **Fork** button at the top right of the repository page.
    This will create your own copy of the repository under your GitHub account.


2. Clone Your Fork

    Clone this repo to your local machine:

    ```bash
    git clone https://github.com/your-username/Awesome_AI4Plant.git
    cd Awesome_AI4Plant
    ```

3. Create a New Branch
    Create a new branch for your contribution:
    ```bash
    git checkout -b add-new-paper
    ```

4. Add Your Contribution

    Edit the appropriate section in the repository.

    Follow the paper format:
    ```html
    **Paper Title**

    Authors. *Journal* **Volume**, Page (Year). DOI

    - One sentence describing the contribution.
    ```

5. Commit Your Changes
    ```bash
    git add .
    git commit -m "Add paper: GenoRetriever for plant TSS prediction"
    ```

6. Push to Your Fork
    ```bash
    git push origin add-new-paper
    ```
7. Submit a Pull Request
    Go to your fork on GitHub and click: `Compare & Pull Request`. Then submit the PR to the main repository.
    Soon We will be merging all your changes into the main branch of this repo. You will get a notification email once the changes have been merged.
